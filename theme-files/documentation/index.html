Installation

Technical Requirements

  • localhost xampp, wamp or any Apache with PHP and MySQL server.
  • Livesite (preferably a sub-domain for testing).
  • For Manual Installation, you need a full installation package for Joomla! 3x.

Quickstart Installation

The Quickstart Package consists of a complete Joomla! + Template + Extensions + Sample Content, excellent for beginner users to explore back-end settings and sample content. Installing the Quickstart is just like installing Joomla!, but you will have the full demo of the template on your server. Once your environment is ready, follow this quick guide:

  • Localhost only: create a new folder in the htdocs folder for Xampp (www folder for Wamp). Example: my-website
  • Unpack/copy all files and folders of the Quickstart folder into this newly created folder or upload them into your sub-domain folder on your live site
  • Open your favorite browser and type in the address bar localhost/my-website for localhost or sub-domain.my-live-site.com for your live site sub-domain, then hit Enter.
  • Install Joomla!. There will NOT be an "Install Sample Data" button. It will be automatically installed.
  • Watch this video tutorial on how to install Dignity Template! Click here!

Manual Installation

Manual Installation refers to installing the template on your Joomla! powered website. This means you already have a website with content items, modules, extensions, so you just decided you want to improve the design of your website. Do not worry, everything is included in the tpl_dignity.zip file and will be automatically installed. So here's what you have to do:

  • Log-in to your Joomla! back-end (www.your-website.com/administrator).
  • Go to Extensions menu at the top and click Extensions Manager.
  • Click Choose File and find the provided installation file tpl_dignity.zip, then click the Open button.
  • Click Upload & Install. The template is now installed.
  • Go to Extensions menu at the top and click Templates Manager.
  • Select the newly installed template and click the Make default button in the top-right toolbar.
  • Check the front-end now. There it is!

Upgrade

Upgrading the template

This stands as a comprehensive and complete guide on how to deal with upgrade on any Joomla! 3.x website on a production live site or any other testing or development environment.

  • Back-up a copy of your template (especially if you have made customizations).
  • Go to Global Configuration and enable the Offline Mode (recommended if you have a production live site, you might not want your users to see layout breaks or critical errors while files are being replaced/copied or edited).
  • Unset the template as default (Extensions menu - Template Manager - select another template and click Default).
  • Uninstall the old version of the template ( For Joomla! 3x -> Extensions - Extensions Manager - Manage tab - select template and click Uninstall).
  • Install the new version of the template.
  • Set the template as default again.
  • Disable Offline Mode in the Global Configuration.

Customizing the template

  • Go to Extensions => Template Manager
  • Select Dignity Template
  • In "Basic Option tab" :
    • field "Logo": Click on "select" change the image logo
    • field "Name Logo": Display Name logo If field Logo is empty
    • field "Select style for template": Select a style for page
    • field "Choose color frontend": Choose a primary color for page. This color will override defalut color of field "Select style for template" selected above. You need Insert color code "#ffd600" if you want display default color of field "select style for template" selected above.

Component ST Page Builder

Have you ever noticed how much time you spend fighting with [shortcodes]? No more trial and errors with “shortcodes magic” – PageBuilder will take care of that for you. You only use your mouse for Drag & Drop so much elements we provided, very friendly and save a lot of time.

In Backend:
  • Go to Component menu at the top and click ST Page Builder.
  • Drop any element on leftbar then Drop to layout aree
    • Element "Column": support grid layout Bootstrap
    • Element "divider": Help you set separation/space for each area in layout
    • Elenemt "HTML": Allows you to create your own HTML using a WYSIWYG editor, additional you can embed Shortcode and HTML Format(see below)
    • Element "Module": You can call a module in Joomla (of course it must publish)
    • Element "GMap": Display Google Map
    • Element "Tabs": Allow display multiple tab
    • Element "Contact Form": You can very easy to make a contact form.
    • Element "Portfolio": Display article from k2
    • Element "Recent Article": Display article from k2
    • Element "Slider": You can make a slider like this http://getbootstrap.com/2.3.2/javascript.html#carousel by Drag and Drop images.
    • Element "Gallery": Like Slider above you can make a Gallery by Drag and Drop images
    • Element "Vimeo": Display embed video from Vimeo
    • Element "Youtube": Display embed video from Youtube
    • Element "Flickr gallery": Display gallery from Flickr
    • Element "FB like Box": Like box facebook
    • Element "Message Box": Display message
    • Element "Services": Display content from K2
  • Then Click on "Save".
  • Goto Menus menu at the top and click Main Menu
  • Click on New,choice menu type "ST Page Builder => Page Builder"
  • Click on Advanced Options tab
  • Select the Item you created before step
  • finally, Click "Save"
